# Canaris EMS - Enterprise Management System

Full Observability Platform with AI-Powered Correlation

## Vision
Hybrid observability platform that correlates network, infrastructure, cloud, applications, and security events to provide real business and service impact visibility.

## Architecture
- **EMS Core** - Platform foundation (NestJS + TypeScript)
- **Network Monitoring** - Existing NMS refactored as module
- **Cloud Monitoring** - AWS EC2/RDS monitoring
- **AI Correlation** - ML-powered root cause analysis
- **APM Lite** - Application performance monitoring
- **ITSM** - Ticketing and SLA management

## Tech Stack
- Backend: Node.js + NestJS + TypeScript
- Frontend: React 18 + TypeScript
- Database: PostgreSQL 15
- Cache: Redis
- ML: Python + FastAPI + scikit-learn
- Cloud: AWS SDK + CloudWatch
- APM: OpenTelemetry
- Infrastructure: Docker Compose
